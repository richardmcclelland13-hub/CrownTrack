# Legal/Data Guardrails

## Approved direction

Use legal source data, generated tiles, licensed providers, and user-owned GPX data.

## Do not use

- Scraped Gaia GPS data
- Scraped Trailforks data
- Scraped Google Maps tiles
- Scraped Garmin map data
- Scraped AllTrails data
- Bulk-downloaded public OpenStreetMap tiles
- Any paid/proprietary map source unless explicitly licensed

## Alberta data direction

Use official Alberta public land, PLUZ, public land recreation, and Crown Land Trails resources where their licence and offline use are confirmed.

## OSM direction

Using OpenStreetMap **data** may be acceptable under ODbL with proper attribution and share-alike awareness. Do not use OpenStreetMap public tile servers for offline map packs.

## UX rule

If legality or source confidence is unknown, the app must show unknown. Never infer legal riding permission from a line existing on a map.
