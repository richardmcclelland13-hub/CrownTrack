# Subagent Playbook — Phase 6

Use subagents/review passes when available.

## Suggested agents

### Mobile UI Engineer
Owns React Native screen/component structure and native-feeling UX.

### Offline Data Engineer
Owns pack registry models, manifests, validation, install states, route coverage.

### GIS Pipeline Engineer
Owns future map-pack builder scaffold, source metadata, tile pipeline docs.

### Legal/Data Auditor
Checks source/licence rules and prevents scraping/proprietary misuse.

### QA Engineer
Writes tests and acceptance report.

## Process
1. Implementer builds small stage.
2. QA agent tests.
3. Legal/data agent checks source assumptions.
4. Mobile agent checks UI regressions.
5. Summarize exact file changes and next step.
