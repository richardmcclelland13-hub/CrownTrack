# Codex Prompt — Stage 1 Native UI Shell

Build the CrownTrack ADV React Native UI shell from the Phase 6 prototype, using mock data only.

## Inputs
- `prototype/crowntrack_phase6_map_pack_pipeline_preview.html`
- `docs/DESIGN_SYSTEM_PHASE5.md` if present
- `docs/PROJECT_STATE_PHASE6.md`
- `codex/AGENTS.md`

## Required screens
- Map
- Packs
- Plan
- Builder or Crew placeholder
- SOS

## Required patterns
- Bottom tab navigation.
- Bottom-sheet style panels.
- Large touch targets.
- Mock map view placeholder, not real MapLibre.
- Mock pack/source/route status chips.

## Do not
- Do not add real GPS.
- Do not add MapLibre.
- Do not use network calls.

## Acceptance
- App runs.
- TypeScript passes.
- UI components are reusable.
- Mock data is centralized.
